プログラム復習コンテンツ

random()関数

サンプルコードの結果

サンプルコード

　　 float speed=2.5;

　　 int diameter=20;

　　 float x;

　　 float y;

　　 　

　　 void setup()

　　 　size(240, 120);

　　 　x=width/2;

　　 　y=height/2;

　　 }

　　 　

　　 void draw(){

　　 　x+=random(-speed, speed);

　　 　y+=random(-speed, speed);

　　 　ellipse(x, y, diameter, diameter);

　　 }

説明文

　random()関数は設定した値の範囲内の乱数を作り出す関数。

　random(a, b);

　a:値の下限　（任意）

　b:値の上限　（必須）

　

　このプログラムは、円がrandom関数によって返された値によって±2.5ずつ移動していくプログラムである。

　background()関数を使用していないため画面がリフレッシュされないため、円の軌道が残り続ける。
